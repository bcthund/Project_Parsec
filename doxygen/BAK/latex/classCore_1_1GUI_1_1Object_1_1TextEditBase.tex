\doxysubsection{Detailed Description}
Keyboard control and \mbox{\hyperlink{classCore_1_1GUI_1_1Object_1_1TextEdit}{Text\+Edit}} object focus tracking. 

Creates a keyboard object to allow input while editing that is used across all instances of \mbox{\hyperlink{classCore_1_1GUI_1_1Object_1_1TextEdit}{Text\+Edit}} objects. Also keeps track of if any \mbox{\hyperlink{classCore_1_1GUI_1_1Object_1_1TextEdit}{Text\+Edit}} objects are currently active preventing others from also being edited at the same time.
