16  /*
17  * USAGE:
18  * Only one copy of this class should exist normally, this primary class is used
19  * by the shaders to render those lights on all scene objects. However a custom
20  * lights class can be supplied with an O3D for example and only those lights
21  * will be used when rendering that object. This would require a copy of the
22  * global light source if that light source is also desired (i.e. sun/moon)
23  *
24  * Each light can have their own diffuse, ambient, and specular colors. Diffuse is
25  * the only color required. If others are omitted then the ambient is calculated
26  * as a percentage below diffuse and the specular as a percentage above diffuse.
27  *
28  * If attenuation is not supplied then a default of Vector3f(1.0f, 0.01f, 0.002f) is
29  * applied. However attenuation is only used for OMNI and SPOT type lights.
30  *
31  * Direction only applies to DIRECTIONAL and SPOT types.
32  *
33  * Position is needed for ALL lights
34  */
