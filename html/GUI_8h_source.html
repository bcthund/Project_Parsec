11 /*
12  * Creates GUI instances for particular objects that can be
13  * used as a building block for a full GUI.
14  *
15  * These classes draw the objects and handle IO, as well as
16  * providing "focus" tracking.
17  * - Focus may be achieved with a static INT that
18  * contains an ID. The ID will represent the
19  * active object.
20  * - Focus should be handled with a "history" so when
21  * an object releases focus the previous object in
22  * history gets focus back.
23  */
